# OAuth2 Identity Service

The EDCs deprecated the OAuth2 Identity Service with version 0.10.0. 
Therefore, the official EDC does not further support OAuth2 as an Identity Service. 
Nevertheless, we moved the latest code from the upstream EDC to our implementation.
We are fully aware that a centralized Identity Service contradicts the dataspace philosophies. 
But to quickly build a dataspace for demonstration purposes, this is a viable option.
